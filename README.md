#### 简介

[TikHub](https://tikhub.io/)是一个All in one的RESTful API平台。

我们提供的API只能获取公开数据，即任何人都可以通过浏览器及APP等访问抖音,TikTok,小红书以获取它们。

如果您有任何建议或者需求，请联系我们，更多的功能正在开发中，敬请期待！
